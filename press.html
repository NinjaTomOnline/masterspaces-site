A quiet utility for people whose Mac desktops keep becoming weather systems.

MasterSpaces helps Mac users snap windows, save layouts, restore project workspaces, and reason about multi-monitor setups from the menu bar.

One-line pitch

MasterSpaces is a direct-download macOS menu bar utility for repeatable window layouts, workspaces, and multi-monitor snapping.

Current version

0.1.0 release candidate. Public launch is pending notarization, stapling, and clean-machine QA.

Platform

macOS 14 or later. Built as a native SwiftUI menu bar app.
