# MasterSpaces public site

This repository contains the public static site exported from the private MasterSpaces app repo.

## Publish

1. Copy or export the hosted site from the app repo.
2. Push this public repo to GitHub.
3. Enable GitHub Pages with the included `Publish Pages` workflow.
4. Confirm the site opens at:
   - Marketing URL: `https://ninjatomonline.github.io/masterspaces-site/`
   - Support URL: `https://ninjatomonline.github.io/masterspaces-site/support.html`
   - Privacy Policy URL: `https://ninjatomonline.github.io/masterspaces-site/privacy.html`

## Before Public Launch

- Verify `support@masterspaces.app` and `privacy@masterspaces.app`.
- Replace canonical URLs and add `CNAME` if a custom domain is used.
- Keep real PNG screenshots in `screenshots/` aligned with the current app release.
- Keep `assets/social-preview.svg`, `assets/social-preview.png`, and `assets/favicon.svg` aligned with the current launch copy.
- Do not add App Store badges until there is a real App Store product URL.
